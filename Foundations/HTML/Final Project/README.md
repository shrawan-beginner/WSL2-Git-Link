Foundations - HTML -Final Project

Project Name: Odin Recipes
Odin Course Link: https://www.theodinproject.com/lessons/foundations-recipes

Project Brief:
This final project will contain all of the HTML tags and elements learned in the HTML lessons of the Foundations course path.

Some of the elements present on the project will be:
<a> - to add link for the recipe page

<h1> - to add a heading
<ul> - to add "Ingredients" list
<ol> - to add "Steps" of the recipe
<img> - to add the final meal image
<p> - to describe the recipes
